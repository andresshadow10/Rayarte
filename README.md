# BIENVENIDOS A **RAYARTE**

**Rayarte** es una papelería creativa donde las ideas cobran vida. Ofrece una amplia variedad de útiles escolares, artículos de oficina, materiales de arte y productos originales que inspiran a escribir, dibujar y crear. Es un espacio pensado para estudiantes, profesionales y amantes del diseño que buscan calidad, color y creatividad en cada detalle
